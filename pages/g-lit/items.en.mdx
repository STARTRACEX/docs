# Items

## base-button

base-button ss a normal button

|propties|type|default|desc|
|---|---|---|---|
|disabled|boolean|false|Disable button|
|ghost|boolean|false|Ghost button|
|color|string|""|Hue|

Use slot elements with names of `pre` and `suf` to specify the front and back elements

If multiple buttons are used, use [`button-group`](./group#button-group)

## base-switch

base-swtch is a normal switch (checkbox input)

|propties|type|default|desc|
|---|---|---|---|
|disabled|boolean|false|Disable input|
|checked|boolean|false|Input checked (Overwrite def)|
|fat|boolean|false|Use fat style (default as rect)|
|def|string|"false"|Checked by default (Will attempt to parse the string)|
|name|string|"checkbox"|Input name|
|value|string|"on"|Input value|

Use slot elements with names of `true` , `false` ,`always` to set display items for different selections

## dialog-item

Dialog-item is an extension of dialog

|propties|type|default|desc|
|---|---|---|---|
|Key | string | "" | Allow Esc key control to close|
|Scale | boolean | false | Allow scaling|
|Model | boolean | false | Whether the default is mode|
|Call | string | "center" | Popup location|

Display by 'show()' by default, 'showModel()' by mode, and hide by 'close()'

## details-item

Details-item is an extension of details

|propties|type|default|desc|
|---|---|---|---|
|open | boolean | false |Open by default|

Change the open status through `toggle ()`, and `isopen()` will return the current open value

If multiple details are used, use [`details group`] (./group # details group)
