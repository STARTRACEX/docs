# layout

## Global layout

### nav-layout

nav-layout contains a default navigation bar, which can contain the footer

|propties|type|default|desc|
|---|---|---|---|
|title|string|""|Subtitle|
|set|number|2|Number of options|
|foo|boolean|false|Disable footer|

If the title is `""` the subtitle will not be displayed, otherwise it will be a string of `|` and subtitle

Set will round down to display the set options

If foo is `true` the bottom(footer) will not be displayed

## Partial layout

### divier-line

divier-line can be used as horizontal and vertical dividing lines

|propties|type|default|desc|
|---|---|---|---|
|before|string|"auto"|Length of before|
|after|string|"auto"|Length of after|
|v|boolean|false|Enable vertical|
