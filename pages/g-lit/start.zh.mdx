# G-lit

这是基于[lit](https://lit.dev)的组件集合

## 快速开始

### 导入文件

将此存储库中包含的js文件导入页面

```html
<!-- 导入一个-->
<script type="module"src="./public/lit/sign-form.js"></script>
<!-- 全部导入-->
<script type="module"src="./public/lit.all.js"></script>
```

**注意** 不要忘记 `type="module"`

### 用于 npm

位于分支`npm`中

**注意** 未发布

## 配置

从配置文件导入`{name}`

然后添加以下代码

```js
/* ... */
name.prefix = "";
name.suffix = "";
```

如果在html中,则需要`type="module"`

```html
<script type="module">
/* ... */
</script>
```
